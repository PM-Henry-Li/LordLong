## 网页小游戏：拼音探险家 (Pinyin Explorer) —— 开发需求与技术文档

### 1. 核心教学逻辑 (Curriculum Logic)

游戏将学习路径分为三个阶段，确保 6 岁儿童从“盲选”过渡到“盲打”。

| **阶段**             | **教学重点**              | **视觉引导**                            | **交互方式**                   |
| -------------------- | ------------------------- | --------------------------------------- | ------------------------------ |
| **阶段一：初识字母** | 认识键盘上的 A-Z 字母位置 | 屏幕显示 1:1 虚拟键盘，对应按键高亮提示 | 单击对应的字母，掉落奖励物品   |
| **阶段二：拼音拼写** | 练习声母、韵母、整体认读  | 字符组合提示（如：b + a = ba）          | 顺序输入声母和韵母，完成拼图   |
| **阶段三：汉字进阶** | 1~6 年级同步生字词        | 显示汉字与拼音，逐渐隐藏拼音提示        | 完整拼写汉字拼音，击败关卡小怪 |

------

### 2. 游戏阶段设计 (Game Mechanics)

#### 阶段 A：字母森林 (入门)

- **玩法：** 屏幕上方掉落带有字母的苹果。
- **引导：** 当字母掉落时，屏幕底部的虚拟键盘上对应的键位会变成金色并跳动。
- **目标：** 帮助孩子建立“字母-键位”的物理反射。

#### 阶段 B：拼音合成室 (进阶)

- **玩法：** 屏幕中心出现一个声母（如 $b$）和一个韵母（如 $a$）。
- **引导：** 孩子需要按顺序按下键盘，完成后屏幕播放语音：“b - a - ba，爸爸的爸”。
- **逻辑：** 按照部编版教材顺序：声母表 -> 韵母表 -> 复韵母 -> 鼻韵母。

#### 阶段 C：词语竞速 (高级)

- **玩法：** 根据小学各年级生字词库，随机出现词语。
- **逻辑：** 第一遍练习显示拼音提示，第二遍练习（复习模式）仅显示汉字。

------

### 3. 激励与激励系统 (Gamification)

- **5 分钟限时：** 设置倒计时进度条，时间到即结算，保护视力。
- **分数计算：** $\text{总分} = (\text{正确次数} \times 10) - (\text{错误次数} \times 5)$。
- **等级勋章：** 累计得分可解锁“拼音学徒”、“键盘侠客”、“汉字大师”等称号。
- **多用户系统：** 简单的本地存储（localStorage）实现。登录界面提供头像选择（如：小猫、小狗），记录每个孩子的最高分。

------

### 4. 补充功能说明 (补齐缺漏)

为了提升专业度，建议增加以下功能：

1. **语音导读 (Audio Feedback)：** 6 岁孩子识字量有限，所有指令需配备清晰的童声语音引导。
2. **指法引导 (Finger Positioning)：** 虽然不强求 6 岁孩子掌握标准指法，但在虚拟键盘上用颜色区分“左手区”和“右手区”是有益的。
3. **防沉迷与视力保护：** 连续游戏两局后，强制弹出“眼睛休息时间”动画。
4. **错误分析：** 记录孩子经常按错的字母，在下一局中提高该字母的出现频率。

------

### 5. 技术架构建议 (Technical Stack)

- **前端：** React.js 或 Vue.js（方便状态管理和组件复用）。
- **动画：** Canvas API 或 PixiJS（确保流畅的掉落效果和粒子特效）。
- **资源库：** * **汉字拼音库：** 使用 `pinyin-pro` 等插件实现汉字自动转拼音。
  - **音效：** 准备正确的“叮”声、错误的“嘟”声以及拼音朗读音频。
